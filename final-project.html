User Test

I did three test to determine what people thought of the front page and how straight forward navigation was. My question test revealed that my idea of having a picture of Chinatown without any context meant that others thought different things about it, which differed from what I thought it represented. This was also affirmed by the five second test of the front page. The click test proved that navigation was not an issue.

Question Test Click Test 5 Second Test

Summary of Findings

Through user research and user interviews I found that my initial thoughts of trying to create a unique portfolio site by giving it a green background and using an image of Chinatown to represent the area from which I lived caused confusion and misunderstanding. Instead of creating something that looked professional, it looked like it was more novice.

One thing that people really liked was the Midnight feature because it was less bright for people's eyes. There were two flaws with the feature however. One flaw was that it wasn't apparently clear to the viewer of it's purpose, and the other was that it did not keep the setting when transitioning to different pages.

Incorporation of Findings

My first task that I felt help the most potential was to make the midnight feature more apparent to the view. To do this, I made it different from other navigation items. Afterwards I wrote a javascript function that would utilize the local storage on the browser to keep the midnight feature throughout the pages as the viewer changed pages.

One thing I realized from the midterm submission was the lack of images and when I put in more images, the page began to lag. This made the photography page less viewer friendly. But it was an easy fix, I found an online resize and compression tool and made all the images uniform in width and compressed the images so they were no more than 400 kb each picture.

I also made time to work on the past projects page so that it would have the "under construction" so that potential hiring managers would be able to view and assess my skills. Even the portfolio would be a part of past/ongoing projects.
